Spring MVC 4 and Swagger API Integration Example
===================

Tools
-------------------
* IDE - NetBeans 8.0.2
* Framework - Spring 4
* Tools - Swagger 
* Tools - Maven

How to Build
-------------------
1. Open project using Netbeans IDE
2. It use Maven, so prepare your internet connection for library downloading
3. Press F6 to Run your project
